#  Live Speech Recognition & Translation  

A sleek **Progressive Web App (PWA)** that performs real-time speech-to-text with instant translation into 20+ languages.  
Designed for accessibility, education, and communication ‚Äî all processed securely in your browser.

---


## [**Try the Live Demo here, click here to see a live demo **](https://live-speech-translation.netlify.app/)

---

##  Features

-  **Real-Time Speech Recognition** ‚Äî Convert spoken words into text instantly.  
-  **20+ Languages Supported** ‚Äî Translate speech across global languages in real time.  
-  **Audio Visualization** ‚Äî See your voice in motion with a live waveform display.  
-  **Download Transcripts** ‚Äî Save conversations or translations as text files.  
-  **Progressive Web App** ‚Äî Works offline, installable on mobile or desktop.  
-  **Privacy-First Design** ‚Äî Everything runs locally in your browser ‚Äî no data is stored or sent to external servers.  

---

##  Technologies Used

- **Web Speech API** ‚Äì Handles live speech-to-text recognition.  
- **Google Translate API** ‚Äì Provides instant translation capabilities.  
- **Web Audio API** ‚Äì Generates live waveform visualization.  
- **PWA (Progressive Web App)** ‚Äì Enables offline access and installation.  
- **Vanilla JavaScript (ES6+)** ‚Äì Lightweight, fast, and dependency-free logic.  
- **Modern CSS3** ‚Äì Responsive and visually clean design.

English, Spanish, French, German, Italian, Portuguese, Japanese, Chinese, Hindi, Arabic, Russian, Korean, Dutch, Turkish  

**Translation (Output):**
